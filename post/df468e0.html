6、数据分析

基础

jupyter 代码编辑器

  • 专门为数据分析工作而打造的编辑器
  • 代码的输入以及代码结果的输出都是在 Cell 中实现的
  • Cell 一共由两个部分组成：In[] 和 Out[]，即输入框和输出框

  • 重新打开学习系统的时候，系统就会“忘记”之前运行过的变量的值。

  • 这个时候就需要点击网页右上角的【运行全部 Cells】按钮，让系统重新“记住”我们所需要用到的变量的值。

pandas 库基础知识

  • pandas 库是一个专门用来解决数据分析问题的库
  • 一般会将 pandas 简化为 pd
  • 速度快以及效率高。

数据结构

Series对象

  • 主要由一组数据及其对应的索引组成
  • dtype: object:其中，dtype 指的是 pandas 库中的数据类型，object 就等价于之前 python 学过的字符串类

创建Series对象

  • 格式：pd.Series()
备注 例子
字典 pd.Series([‘赵’, ‘钱’, ‘孙’, ‘李’]）
标量 pd.Series(5)
n维数组 import numpy as np;pd.Series(np.array([1,2]))
字符串 pd.Series(“hello “)

DataFrame 对象

  • DataFrame对象是一种表格型的数据结构，包含行索引、列索引以及一组数据。

创建DataFrame对象

  • pd.DataFrame(data)
  • 当我们给参数 data 传入字典的时候，字典的键会变成 DataFrame 对象的列索引，字典的键所对应的值会变成 DataFrame 对象的数据。
  • 传入的字典的各个键所对应的值长度一定相等；否则会报错
  • 对象[‘键名’]：通过列索引——–是Series对象
  • n 维数组的方式
  • 分析流程：明确目标、数据处理、数据分析、数据展现以及报告撰写

读取数据

  • pd.read_csv(path, encoding) 就是其中一种高效读取 csv 文件的方法，返回的是一个 DataFrame 对象。

数据处理

数据清洗

  • 数据处理步骤：数据清洗、数据整理、数据写入

  • 在 pandas 库中，NaN 代表缺失值，意思是这里的数据是空的。

处理缺失值

  • **df.info()**：可查看数据情况–数据总结

查找缺失值
  • isna() 方法来查找 DataFrame 对象以及 Series 对象中的缺失值
  • 缺失数据会用 True 来表示，False 则代表这里的数据不缺失
  • ，还可以使用df.head()或df.tail()方法查看数据。
  • DataFrame 对象使用 df.head() 方法默认可以查看数据的前 5 行，df.tail() 方法则默认可以查看数据的后 5 行；
  • df.head(i)中i代表常看前i行；df.tail(i) 中i代表常看后i行
  • NaN 代表的就是缺失数据
删除缺失值
  • 只要执行 df.dropna() 这一代码方法，就可以将 DataFrame 对象中包含缺失值的每一行全部删掉
  • 刚才我们运行的 mask_data.dropna() 代码返回的是一个删掉所有缺失数据的 DataFrame 对象，但这并不意味着它修改了 mask_data 数据。
  • 我们需要将 mask_data.dropna() 的运行结果重新赋值给 mask_data 变量，这样就可以将运行结果保存下来
  • 针对某几列的缺失数据进行删除
    • 对象。dropna(subset=[‘所删除列名’])
    • test_data 中’品牌’、’上牌时间’、 ‘里程数(km)’和 ‘保值率’四列数据缺失的行，可以如下书写代码：
填充缺失值
  • DataFrame对象和Series对象都可以使用**fillna()**的方法给缺失值填充数据。
  • 只需要将要填充的数据作为参数写在括号内，就能把所有缺失值都替换成括号里的数据
  • 有些缺失数据我们最好不要删除，因为它们依然有很强的分析价值，这时会用填充缺失值。

处理重复值

  • 查找重复行
    • 使用 df.duplicated() 方法会返回一个 Series 对象，找出所有重复值。重复为 True，不重复为 Falsel
  • 可以将 **mask_data.duplicated()**返回的结果放入中括号 [] 中，用来索引 mask_data 数据，查看 mask_data 数据中重复的行。
  • 可以使用 pandas 库的 df.drop_duplicates() 方法直接删除 DataFrame 对象中重复出现的整行数据。
    • df.drop_duplicates() 方法删除的也都是重复出现的行，因此所有重复数据的第一行都会保留

处理异常值

  • 检查异常值 describe()
    • describe() 方法返回出来的统计信息分别代表数值型数据的频数统计、平均值、标准差、最小值、第一四分位数、中位数、第三四分位数以及最大值。
    • e+n 代表的是 10 的 n 次方

  • 抽取数据范围

    • 单价一列小于等于 200 的数据，就可以设置条件表达式 mask_data[‘单价’] <= 200

数据整理

  • 指的是我们在数据分析前对所需字段进行数据排序、数据转换、数据抽取、数据合并、数据计算等准备操作
  • 字段指的是表格的列，像我们这次分析的数据就包含 ‘订单编号’、’日期’、’省’……等字段。

datetime

  • 专门储存时间日期的数据类型，可以直接从中提取想要的时间单位
转换日期数据
  • pandas 库中我们可以使用 pd.to_datetime(arg, format) 来将 DataFrame 对象或者 Series 对象的数据类型转换成 datetime 类型。
  • arg 参数为我们要转换的数据，它可以是 DataFrame 对象或 Series 对象
  • mask_data 中的’日期’字段，所以对应的 arg 参数就是 mask_data[‘日期’]。
  • format 参数为 datetime 类型的日期格式，比如说这份数据，它是以年-月-日的形式出现的，那么它对应的 format 就是 ‘%Y-%m-%d’（year-month-day的缩写）。
  • 人提取日期信息
    • 获取这份数据的年、月、日的信息，可以通过 Series.dt.year、Series.dt.month以及Series.dt.day来获取
  • 添加新的一列
    • df[‘colname’] = Series的方式来为原数据添加新的一列：colname 指的是要添加的新列的列名

数据写入

  • df.to_csv(‘路径’，编码)方法
  • 可以生成新文件
  • pandas 库将 csv 文件读取为 DataFrame对象 时都会自动生成行索引。；默认值为 True，会让你写入的 csv 文件包含一列行索引
  • 把这个参数设置为 index = False 就可以取消写入行索引，不用将行索引那一列写入 csv 文件。

总结

数据分析

分组聚合

  • 分组聚合操作指的是按照某项规则对数据进行分组，接着对分完组的数据执行总结统计的操作（求和、平均值等）

单层分组聚合操作

  • df.groupby(by)[‘列索引’].mean()
  • 返回Series对象
  • 单层分组操作
    • 分组操作：dfgroupdy(by)
    • 参数 by 代表了我们想要对哪一列数据进行分组操作
  • 聚合操作
    • [‘列索引’].mean
    • ‘列索引’指的是需要对哪一列数据进行聚合操作
    • mean() 指的是求平均值

常见聚合操作

流程 方法 示例
求最小值 min() 一组数据最小值
求最大值 max() 一组数据的最大值
求标准差 std() 一组数据的标准差
求平均值 mean() 一组数据的总和除以一组数据的个数
求中位数 mendian() —组数据中处于中间位置的数
求总和 sum() 一组数据的总和
求数量 count() 一组数据的非空数据的个数
求频数分布 value_counts() 一组数据各个类别出现的次数

多层分组聚合操作

  • 格式：df.groupby(by)[‘列索引’].mean()。
  • 多层分组聚合操作需要传多个列索引给参数 by

s.unstack() 函数

  • 可以将一个多层分组聚合后的 Series 对象转变成 DataFrame 对象
  • 作用就是将其索引的最后一列转变成 DataFrame 对象的列索引，而剩下的索引则转变成 DataFrame 对象的行索引。

绘制单条折线图

  • s.plot()
参数 说明 参数值类型（常用） 示例
kind 设置图表类型 字符串 kind=’line’

多条折线图

  • 多条折线图是针对一个 DataFrame 对象来绘制的，而单条折线图是针对一个 Series 对象来绘制的。
  • df.plot()
  • 观察折线图，有三个要点，一是整体的走势，二是走势的规律性，三是走势的波动。

其他图形拓展

饼图

  • 主要可以用来反映数据的占比情况
  • 函数plt.pie()函数

条形图

  • 主要可以用来反映不同数据值的差异性。

matplotlib

  • 从应用范围上来看，在 python 数据分析领域，matplotlib 库是与 numpy 库、pandas 库并驾齐驱的三库之一，被称作 python 数据分析的“三剑客”，由此可见其应用范围之广。
  • 个完整的图形除了图表，还包括一系列的图表元素：图表标题、坐标轴刻度、坐标轴标题、图例以及数据标签。

常用函数

设置画布大小 ：plt.figure()

  • 元组形式:plt.figure(figsize=(6,6))

对画布进行保存：plt.savefig()

  • plt.savefig(‘路径’)

绘制折线图

  • 颜色

  • marker参数

  • plt.plot()函数
    • color 参数表示图表的颜色
    • plt.plot(x,y,color=’颜色’)

柱状图绘制

  • plt.bar()函数
    • plt.bar()的参数是 x 和 height，height 表示柱子的高(对应 y 坐标值）； alpha 参数表示图表的透明度
    • plt.bar(x,height=y,color=’颜色’,alpha=c)
    • c为0到1

设置图标标题，坐标轴

plt.title()函数

  • plt.title(lable,fontsize=None)
  • label表示标题，为字符串类型；fontsize为字体大小，为数值类型
  • 还可用fontdict代替fontsize

  • width 则需要传入(0, 1]区间 (大于 0，小于等于 1) 内的浮点数，数值越大表示柱子的宽度越宽；当 width 的值为 1 时，柱子之间的间隙就为 0。
  • 参数 alpha可以设置柱子颜色的透明度，需要传入(0, 1]区间内的浮点数，数值越小，表示柱子颜色越浅；相反，则越深。适当地调整颜色深浅，可以提升视觉舒适度。

设置坐标轴刻度

  • 坐标轴刻度，设置 x、y 轴刻度的函数分别是 plt.xticks()和 plt.yticks()。
  • 调用这两个函数，就可以根据 x、y 坐标值自动生成 x、y 轴刻度字体大小

坐标轴标题

设置图例

  • 设置图例用到 **plt.legend()**函数，其中的参数labels表示图例名称，对应图中的多个条件，通常传入可迭代对象，比如列表、Series 对象等。

设置数据标签

  • 调用 plt.text()函数可以在图表的指定位置添加文本，但是每次只能添加一个。
  • 这里需要重点强调 x、y 的用法，x、y 表示所添加文本的位置，受到函数本身的属性限制，常见为数值或者字符串类型，这与绘图函数中 x、y 可以传入可迭代对象有很大的差异。

  • zip（）函数

    • zip(x,y)

    • 你可以将 zip()函数理解为拉链，x=(x1, x2, x3,……, xn)为拉链的左边，y=(y1, y2, y3,……, yn)为拉链的右边，经过 zip()函数，就合并为(x1, y1), (x2, y2), (x3, y3),……,(xn, yn)。

    • zip()函数可以从 x、y 元素序列中依次取出元素，按照顺序进行一对一匹配。

Series 对象排序

  • 在 pandas 中，我们可以使用s.sort_values()对 Series 对象的数值型数据进行排序。
  • 默认为升序排列（从小到大），将 ascending 参数设置为 False 就可以让它降序排列（从大到小）。

Apriori 算法

设置列索引值

  • 以字典作为数据传入时，字典的键会作为 DataFrame 对象的列名显示，而字典的值会作为对象的列数据显示。
  • 以嵌套列表作为数据传入时，列表的元素会作为 DataFrame 对象的行数据显示，且会为数据默认生成从 0 开始的列名。

  • 参数 columns 的值可以是一个列表，该列表的长度需与传入 DataFrame() 的列表的元素长度一致

  • 已创建的DataFrame对象列表名修改

数据排序

sort_values(by) 方法

  • 该方法可以对指定列进行排序操作，默认升序排列;by可为列名

重置行索引

  • 重置索引可以使用 reset_index() 方法
    • drop的值为False时保留原来的索引；drop的值为True时重置索引（从0开始）

数值的四舍五入

  • round()

去除数据单位
